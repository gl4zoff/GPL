Value types:

int (int [Name] = [value])

string (str [Name] = [value])



Functions:

write (write [Value type's name] or ['value'])

V - size (V: [value]) (Nuber of strings) (Default: 100)
...



Example:

V: 13
int sNum = 54
int fNum = 24
str hw = Meow

write hw
write sNum

sNum - fNum
sNum / 3

write sNum
write 'Hello World!'

(Output: 54  10  Hello World!)
