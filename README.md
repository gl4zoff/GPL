Value types:

int (int [Name] = [value])

string (str [Name] = [value])



Functions:

write (write [Value type's name] or ['value'])

V - size (V: [value]) (Nuber of strings) (Default: 100)

if (if [first value] (== or != or < or >) [second value]{})
...

V: 20
int sNum = 54
int fNum = 24
str hw = Meow

if sNum > fNum

{

write hw

}

sNum - fNum
sNum / 3
